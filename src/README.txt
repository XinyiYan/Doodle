End button will be disabled if the slider position is already at the end or there is nothing to display

Play/Start(Playback) cannot be performed at the same time

Support saving and loading binary file

Support scrubbing

Contents in canvas will resize itself to fit available spaces


Enhancements

1. Customizable color palette:
    Users can determine what colors the color palette displays by right clicking one of those six color buttons.
    Right clicking will invoke a JColorChooser to select a custom color which will be displayed on the color palette

2. Ability to play animations both forward and backward
    Play button can play animations from the current slider position to the end.
    If the current slider position is already at the end, play button will play animations from the begining.

    Start button will play animations backward and rewind to the start.
